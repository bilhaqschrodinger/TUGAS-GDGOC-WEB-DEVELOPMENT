# TUGAS-GDGOC-WEB-DEVELOPMENT

**Mini Project Web Development GDGOC**

Proyek ini adalah tugas mini project dari GDGOC yang berfokus pada pengembangan dasar web menggunakan teknologi **HTML**, **CSS**, dan **JavaScript**.

## üöÄ Fitur Utama

Proyek ini terdiri dari beberapa halaman dasar yang dibangun untuk memamerkan keahlian dasar dalam pengembangan web:

  * **`index.html`**: Halaman utama (Home).
  * **`about.html`**: Halaman tentang diri atau proyek.
  * **`education.html`**: Halaman yang menampilkan riwayat pendidikan.
  * **`form.html`**: Halaman yang berisi formulir input untuk interaksi pengguna.
  * **`style.css`**: Berkas CSS yang mengelola tampilan dan gaya visual semua halaman.
  * **`script.js`**: Berkas JavaScript yang menangani fungsionalitas dan interaktivitas halaman web, terutama untuk halaman formulir.
